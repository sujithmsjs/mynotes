2. platform as a service (PaaS) and packaged
3. software as a service (SaaS).

Amazon launched AWS, a cloud computing platform to allow the different organizations to take advantage of reliable IT infrastructure.

AWS Account Identifiers
________________________
AWS assigns two types of unique ID to each user's account:

1. An AWS account ID: AWS account ID is a 12-digit number such as 123456780123 which can be used to construct Amazon Resource Names (ARNs).

2. A canonical user ID: A Canonical user ID is 64-digit hexadecimal encoded a 256-bit number.

AWS account ID is a 12-digit number such as 123456780123. A canonical user ID is used in an Amazon S3 bucket policy for cross-account access means that AWS account can access the resources in another AWS account.
